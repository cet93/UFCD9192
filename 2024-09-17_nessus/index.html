2024-09-17 Nessus

Kali - Fazer download do Nessus

No Kali, navegar até ao site: https://www.tenable.com/products/nessus/nessus-essentials Efetuar o download do Nessus Essentials

Após fazer o registo, proceder ao download.

Selecionar a plataforma Debian - amd64, selecionar Download.

Kali - Instalar o Nessus

No terminal, navegar até à pasta onde foi feito o download do Nessus.

sudo su

cd Downloads

Executar o comando "dpkg -i Nessus-8.15.0-debian6_amd64.deb"

Kali - Iniciar o Nessus

Executar o comando "systemctl start nessusd"

Executar o comando "systemctl enable nessusd"

Executar o comando "systemctl status nessusd"

Kali - Aceder ao Nessus

No browser, aceder a https://localhost:8834

Selecionar Continue.

De seguida selecionar "Register for Nessus Essentials" e selecionar Continue.

Nessus formulário

Não é necessário registar se já efetuámos o registo no site. Neste caso selecionamos Skip Preencher o formulário e selecionar "Register".

Nessus - Activation code

Copiar o Activation code e colar no campo "Activation code" e selecionar "Continue".

Nessus - Criar conta de utilizador

Criar uma conta de utilizador e selecionar "Submit". (Se der erro, tentar novamente, sem ser através da firewall)

Nessus - Download Plugins

Esprar que o NEssus faça o "Download Plugins".
