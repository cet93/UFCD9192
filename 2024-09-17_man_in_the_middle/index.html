Kali - Identificar o IP

Digitar o comando "ip a" para ver a configuração IP do Kali.

Kali - Ettercap

Como root, executar o camando "ettercap -G", para abrir a aplicação em modo gráfico. Com o rato clicar no visto, para selecionar a interface de rede.

No menu selecionar "ARP poisoning".

E clicamos em OK.

No menu dos 3 pontos, selecionamos "Hosts\Scan For Hosts".

Voltamos ao menu dos 3 pontos, selecionamos "Hosts\Hosts List".

Selecionamos o IP (192.168.59.254) do PFSENSE e clicamos em Targe1, de seguida selecionamos o IP (192.168.59.1) do Windows Server e clicamos em Target2.

No Windows Server, fazemos login com o utilizador "admin".

No Ettercap, podemos ver as conexões, se formos ao menu dos 3 pontos, selecionarmos View\Connections.

Se fizermos duplo clique sobre a conexão entre o Windows Server e o PFSENSE, na porta HTTPS (22000), confirmamos que a ligação se encontra encriptada.
